## Tech Stack

- **HTML5** вЂ” Semantic structure
- **CSS3** вЂ” Custom styles + Bootstrap 5.3
- **JavaScript (Vanilla)** вЂ” Dynamic components, search, cart, wishlist
- **Font Awesome 6.5** вЂ” Icons
- **LocalStorage** вЂ” Cart, wishlist & user session persistence

---

## Features

- Live Search вЂ” Instant product search overlay with thumbnails
- Shopping Cart вЂ” Add/remove items, quantity control, persistent via localStorage
- Wishlist вЂ” Save favourite products
- Login / Logout вЂ” Simple auth flow with user avatar in navbar
- Fully Responsive вЂ” Mobile-friendly hamburger menu & layout
- Filter & Sort вЂ” Filter by category, sort by price on shop page
- Order Tracking вЂ” View order status page

## Getting Started

No build tools required. Just open in a browser:

```bash
# Clone the repo
git clone https://github.com/truonghuy170805-byte/FRMNT_CLOTHING.git

# Open index.html in your browser
open index.html
```

Or use a local server for best experience:

```bash
npx serve .
```
